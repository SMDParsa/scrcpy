# scrcpyfs
A usefull feature to enhance scrcpy

## What is scrcpy?
Its an application to cast your android phone screen into your computer. you can download it from github with this link:
https://github.com/Genymobile/scrcpy.git

## What this file do?
This file is created to launch scrcpy in full screen withoud any boundaries and windows titlebar.

## How to use it?
Simply download the file and place inside scrcpy folder and double click it.
it will open scrcpy in full screen for you.

watch full tutorial on YouTube:
